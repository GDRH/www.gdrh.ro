Alderon Needs Your Help

The Project "Alderon needs your help" is a 3D game that can be described as as a Space Shooter where you
control a spaceship and you have to destroy enemies.

How will the game be realized?

The project will be realized with the help of:
  1. Unity Game Engine for creating and compiling the game.
  2. For the scripts we will use Microsoft Visual Studio in C# using the Unity library and other libraries offerd by Unity.
  3. For the 3D models we will use Blender.
  4. If we have time, for the music we will use Fruit Loop Studio.

How will the final project look?

The game scene will be set in space. You control a spaceship and with the help of your allies you have to destroy enemy spaceships. Note: Your allies amd the enemies are controlled by an AI.
If we have time, after we finish the base game we will, in this order :
  1. Make the AI difficulty adjustable.
  2. We will create music for the game.
  3. We will try to make the game multiplayer/Online.
  4. Diffrent ship classes, like scout ships, gun ships.
